  • ID - Parenka elementa kuris turi nurodyta id (#IDpavadinimas)

Spalvos

  • Spalvos pavadinimas - nurodai spalvai paskirta pavadinima
  • RGB - nurodai RGB koda

Yra 140 skirtingu spalvu pavadinimu. (red, crimson, azure)

Red green blue - RGB . Gali turet reiksmes nuo 0 iki 255. Norit pridet transparency - reikia pridet Alpha nustatyma. (0,255,10 ; rgba - 2,255,10,0.564)

Hexidecimal (16'ainis kodas) gali turet reiksmes nuo 00 iki ff. Norint prideti transparency , prie hex reikia prideti papidloma nurodyma kuris eina nuo 00 iki ff (#00ff0a | #00ff0a90)

Display

  • Inline - Linijinis
  • Block - blokinis
  • Inline-Block - linijins blokinis

Linijinis elementas uzima tik tiek vieots kiek reikalauja jo viduje esantis contentas. Linijiniam elementui negalima nustatyti dydzio ( width/height).

Blokinis elementas uzima visa eilute. Blokiniam elementui galiman nustatyti dydi (width/height)

Linijinis- blokinis elementas uzima tiek kiek reikalauja jo viduje esantis content , arba yra nurodyto dydzio.
