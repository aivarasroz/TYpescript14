Paveldejimas

Is tevinio elemento, vaikiniai elementai paveldi tam tikrus stilius. Paveldimi stiliai dazniausiai buna susije su: color, font-size, font-family. Nepaveldimi stiliai yra: width, height, display, border, fonai.
