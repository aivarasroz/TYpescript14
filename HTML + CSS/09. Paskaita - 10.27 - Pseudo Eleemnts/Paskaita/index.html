Selectors

  • element - nurodytas elementa (tag'a)
  • .class - elementas su nurodyta klase
  • #id - elementas su nurodytu id
  • * - viskas
  • selector selector - vieno selektoriaus viduje esantis kitas
  • selector > selector - elemento vaikinis elementas iskart pats pirmas , vienu lygmeniu zemiau
  • selector, selector = abiem selektoriam nurodomi stiliai vienu metu
  • SelectorSelector - elementas turintis abu/visus nurodytus selektorius
  • selector1 + selector2 - uz vieno elemento einantis kitas elementas
  • Selector1 ~ Selector2 - elementas einantis uz kito elemento ( tam paciam lygmenyje)

Pseudo selectors

  1. :link - nelankyta nuoroda
  2. :visisted - lankyta nuoroda
  3. :hover - kai peles kursorius uzejes ant elemento
  4. :active - paspaudimo momentas
  5. :empty - tuscias elementas (be nieko)
  6. :fullscreen - elementas esantis pilno ekrano busenoje
  7. :not (selector) - nurodo. kuris elementas neturi kazkokio stiliaus
  8. :root - parenka dokumento sakninius elementus
  9. :target - [er nuoroda parinktas elementas
  10. :first-child - pirmas vaikinis elementas
  11. :last-child - paskutinis vaikinis elementas
  12. :only-child - vienintelis vaikinis elementas
  13. :first-of-type - to paties tipo elemento pirmas variantas
  14. :last-of-type - to paties tipo elemento paskutinis variantas
  15. :nth-child(nr) - kelintas vaikinis elementas
  16. :nth-last-child(nr) -
  17. :nth-of-type -
  18. ::before - pries turini esantis turinys
  19. ::after - po turinio esantis turinys
  20. ::first-letter - pirmoji elemento raide
  21. ::first-line - pirmoji elemento eilute (kuri atvaizduojama ekrane)
  22. ::marker - Saraso elementu zymeklis
  23. ::selection - pazymeta informacija.
