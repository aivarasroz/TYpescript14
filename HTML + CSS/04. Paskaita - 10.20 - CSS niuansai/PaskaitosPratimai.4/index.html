Papildomi poziciju stiliai

Prisikiriant elementui bet kuria pozicija apart static, elementas igauna galimybe naudotis siais stiliais: top/bottom/left/right/z-index.
Top/bottom/left/right priklausomai nuo suteiktos pozcijos rodo elemento pozicija puslapyje.
O z-index rodo elemento sluoksni puslapyje.

Statinis elementas neigauna jokiu stiliu ir negali naudoti posiziciju.
